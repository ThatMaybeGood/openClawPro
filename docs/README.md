# 筑栖平台 - 安心建造，智慧安居

## 🏠 平台简介
筑栖是一个连接用户与建筑供应商的综合服务平台，提供从设计到入住的全流程服务。我们致力于让房屋建造过程透明化、智能化、安心化。

## ✨ 核心功能

### 1. 📐 户型上传与设计
- 支持户型图/宅基地信息上传（JPG/PNG/PDF）
- AI 辅助设计方案生成
- 精准预算报价系统
- 3D 效果预览（开发中）

### 2. 👷 施工团队选择
- 严选认证施工团队
- 资质等级公示
- 完成项目数展示
- 用户评价系统
- 多对一比较选择

### 3. 🔨 装修材料供应
- 对接优质建材供应商
- 价格透明对比
- 品质保证体系
- 配送直达工地
- 在线选购追踪

### 4. 📊 全流程施工监控
**环境监测：**
- 实时天气数据
- 温湿度监测
- 风速/空气质量

**进度管理：**
- 施工进度实时更新
- 照片/视频记录
- 关键节点提醒

**质量管控：**
- 质量问题上报
- 整改跟踪记录
- 验收确认流程

### 5. 🎨 专业设计服务
- 资深设计师一对一
- 个性化空间方案定制
- 平面布局 + 效果图
- 施工图制作

### 6. 💰 精准预算管理
- 分项明细报价（材料费/人工费/设备费等）
- 费用透明可控
- 合同锁价
- 变更审批流程

## 🚀 快速开始

### 本地开发
1. 克隆仓库：`git clone git@github.com:ThatMaybeGood/openClawPro.git`
2. 进入项目目录：`cd zhiqu-platform/src`
3. 双击打开 `index.html` 查看效果

### 部署上线
1. 构建生产版本
2. 配置 Web 服务器（Nginx/Apache）
3. 域名解析和 HTTPS 配置
4. 访问正式上线
